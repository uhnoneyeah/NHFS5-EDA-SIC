# 🏥 NFHS-5 India Health Indicators Analysis

This project analyzes NFHS-5 state-level health data to study obesity, anaemia, hypertension, and blood sugar patterns across India.

---

## 📌 Objective

- Clean and structure NFHS-5 dataset
- Compare health indicators across states and genders
- Analyze correlations between malnutrition and lifestyle diseases
- Visualize regional health disparities

---

## 🧰 Tech Stack

- Python
- Pandas
- Matplotlib
- Seaborn
- Folium

---

## 🧪 Workflow

### Phase 1: Data Setup
- Loaded NFHS-5 dataset
- Selected key health indicators
- Cleaned missing/non-numeric values
- Created **Gender Obesity Gap**

### Phase 2: EDA
- Correlation analysis (Anaemia vs Obesity)
- Bar chart (Hypertension comparison by gender)
- Scatter plot (Anaemia vs Obesity in women)

### Phase 3: Geospatial Analysis
- Built India map using Folium
- Created choropleth for obesity rates
- Added state-level interactive markers

---

## 🧠 Key Insight

India shows strong regional variation in obesity and anaemia, along with clear gender-based health differences.
